# Roadmap

This roadmap is intentionally scoped to reusable, data-free OSS assets.

## 0.1.x: Maintainer Foundation

- Add CI for all pull requests.
- Add issue and pull request templates.
- Expand synthetic workflow examples.
- Document release and privacy checks.

## 0.2.x: Multilingual Workflow Vocabulary

- Add English, Chinese, and Thai terms for customs jobs, declarations, outsourced transport, finance, documents, and compliance.
- Add terminology review guidance for maintainers.
- Keep translations generic and source-reviewable.

## 0.3.x: Document Readiness Rules

- Expand document-readiness contracts for clearance, release, and closure.
- Add examples for missing invoices, unverified documents, and declaration rejection paths.

## 0.4.x: Maintainer Automation

- Add examples for PR review prompts, issue triage summaries, and release-note generation.
- Keep automation examples explicit about privacy, authorization, and synthetic data.
- Add more deterministic checks before calling any LLM or API model.

## Not Planned

- Production database dumps
- Customer portals
- Private business workflows
- Owned-fleet dispatch systems
- Legal advice or jurisdiction-specific compliance guarantees
